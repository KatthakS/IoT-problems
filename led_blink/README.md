Problem Statement: Display Hello World on OLED screen and blink the LED lights on the Heltec WiFi LoRa 32 board.
Code Editor: Visual Studio Code for macOS
Code Builder and library Manager: PlatformIO extension on VSCode
Framework: Arduino
Library Installed: U8g2


Steps:
1. Install Visual Studio Code: https://code.visualstudio.com/download
2. Install PlatformIO: https://docs.platformio.org/en/latest/ide/vscode.html#ide-vscode
3. Install the library to interface with the OLED screeen of Heltec board by clicking on the PlatformIO library registry. 
4. Add a simple code to upload to the Heltec board (i.e. main.cpp).
5. Click on the upload button at the bottom of VSCode window.
6. There you go!!
